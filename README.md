# Weather Styling Application (Flutter Demo)

## Prerequisite
- URL, Endpoint, API Key 는 환경변수로 설정
- `assets/env/.env` 파일 생성
    ```sh
    BASE_URL= # 공공데이터포털
    ENDPOINT= # 기상청_단기예보 ((구)_동네예보) 조회서비스
    API_KEY=  # 일반 인증키 (Encoding)
    ```
